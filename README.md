# Algorithm Visualizer

A simple, browser-based algorithm visualizer for sorting and pathfinding. It includes interactive controls to generate data, start animations, adjust speed and size, and reset the view.

## Features
- Sorting visualization (Bubble Sort, Merge Sort)
- Pathfinding visualization (Dijkstra, A* Search)
- Adjustable speed and size controls
- Live stats for comparisons, swaps, and execution time
- Pathfinding grid set to 7x7 for easy viewing

## Project Structure
- index.html: Main page markup
- style.css: Styles for layout and visuals
- script.js: Visualization logic and interactions

## Run Locally
Open index.html in any modern browser.

## Usage
1. Choose a visualizer mode (Sorting or Pathfinding).
2. Select an algorithm.
3. Adjust speed and size as needed.
4. Click Generate to create data or a grid.
5. Click Start to run the visualization.
6. Click Reset to regenerate and clear stats.

## Notes
This project is intended for learning and demonstration purposes.
